**Qilin Ransomware Actively Exploiting False Human Verification**

**Fake WhatsApp API on npm Steals Developer Data**
A malicious package disguised as a WhatsApp API client has been discovered on the npm software registry. This fake package contains code that steals SMS messages, contact lists, and login tokens from developers who install it in their projects.
Why it Matters: This is a software supply chain attack. If a developer uses this malicious package, the data theft could extend to the applications they build and the users of those applications, creating a widespread data breach.
