**Critical Remote Code Execution Vulnerability in BeyondTrust Products**
A critical remote code execution (RCE) vulnerability has been discovered in BeyondTrust's RS and PRA products. The flaw allows an attacker to execute arbitrary code on a vulnerable system without any authentication or user interaction. BeyondTrust has released patches for the affected products (RS versions 25.3.1 and prior, and PRA versions 24.3.4 and prior).
Why it Matters: This is a critical vulnerability that could allow a complete takeover of affected systems. The lack of authentication required makes it easily exploitable. Organizations using these BeyondTrust products should patch their systems immediately to avoid potential compromise, data exfiltration, and service disruption.
Link: https://www.securityweek.com/beyondtrust-patches-critical-remote-code-execution-vulnerability/

**Critical SQL Injection Vulnerability in FortiClientEMS**
Fortinet has released security updates to address a critical SQL injection vulnerability (CVE-2026-21643) in its FortiClientEMS software. The vulnerability, rated with a CVSS score of 9.1, could allow an unauthenticated attacker to execute arbitrary code or commands via specially crafted HTTP requests.
Why it Matters: This is another critical vulnerability that could lead to a full compromise of the affected system. The fact that it is an unauthenticated SQL injection makes it a high-priority target for attackers. Enterprises using FortiClientEMS should apply the security updates as soon as possible.
Link: https://thehackernews.com/2026/02/fortinet-warns-of-critical-forticlientems-sql-injection-vulnerability.html

**Widespread Campaign Targeting Cloud-Native Environments**
A threat actor group known as TeamPCP (and other aliases) is conducting a large-scale campaign targeting cloud-native environments. The attackers are exploiting exposed Docker APIs, Kubernetes clusters, Ray dashboards, and Redis servers, as well as the recently disclosed React2Shell (CVE-2025-55182) vulnerability. The goal of the campaign is to build a malicious infrastructure for further attacks.
Why it Matters: This campaign highlights the significant risks associated with misconfigured cloud environments. Organizations using cloud-native technologies should review their configurations, secure exposed endpoints, and patch any vulnerable components to protect themselves from this ongoing threat.
Link: https://thehackernews.com/2026/02/massive-campaign-targets-cloud-native-environments-to-build-malicious-infrastructure.html

**Critical Vulnerabilities Patched in SAP and Ivanti Products**
SAP and Ivanti have both released critical security updates for their products. SAP's February 2026 Security Patch Day addressed critical code injection vulnerabilities in SAP CRM and SAP S/4HANA. Ivanti released patches for two new vulnerabilities in its Endpoint Manager (EPM) platform that could allow remote attackers to leak arbitrary data.
Why it Matters: These vulnerabilities affect widely used enterprise software and could lead to significant data breaches or system disruption if exploited. Customers of SAP and Ivanti should review the security advisories and apply the necessary updates to their systems.
Link: https://www.cybersecuritynews.com/sap-and-ivanti-patch-critical-vulnerabilities/

**Deceptive 7-Zip Campaign and Other Malware Threats**
A deceptive campaign is tricking users into downloading a fake version of the 7-Zip file archiver, which turns their computers into proxy nodes for the attacker's use. Other new malware threats include the ILOVEPOOP Toolkit, which exploits the React2Shell vulnerability, and new tools from the APT36 hacker group targeting Linux systems.
Why it Matters: These threats highlight the ongoing need for user awareness and caution when downloading software from the internet. They also show the diverse range of tools and techniques used by attackers to compromise systems.
