**Critical Fortinet Vulnerability Actively Exploited**
A critical authentication bypass vulnerability, identified as CVE-2026-24858, is being actively exploited in the wild. The flaw affects Fortinet's FortiOS, FortiManager, and FortiAnalyzer products. If FortiCloud Single Sign-On (SSO) is enabled, an attacker with a FortiCloud account could potentially log into other registered devices. Fortinet has released patches and urges customers to upgrade immediately.
Why it Matters: This is a critical vulnerability that could allow an attacker to gain unauthorized access to an organization's network. The fact that it is being actively exploited means the risk is immediate. Any company using the affected Fortinet products should prioritize applying the patch.
Link: https://thehackernews.com/2024/02/fortinet-warns-of-new-critical-flaw.html

**Russian State-Sponsored Hackers Exploit Microsoft Office 1-Day Flaw**
The Russian advanced persistent threat (APT) group known as APT28 or Fancy Bear has been observed using a newly discovered vulnerability in Microsoft Office (CVE-2026-21509) in a recent espionage campaign. The group weaponized the flaw within 24 hours of its public disclosure, targeting European government and military entities with spear-phishing emails.
Why it Matters: This incident highlights the speed at which sophisticated threat actors can operationalize a newly disclosed vulnerability. It underscores the importance of applying security patches as soon as they become available, especially for widely used software like Microsoft Office.

**SolarWinds Web Help Desk Flaw Added to CISA's Actively Exploited List**

The flaw, CVE-2025-40551, is a remote code execution vulnerability that can be exploited by an unauthenticated attacker.
Why it Matters: The inclusion of this vulnerability in the KEV catalog indicates that it is being actively exploited in the wild. Organizations using SolarWinds WHD should immediately apply the available patches or take mitigation steps to avoid a potential compromise.
Link: https://thehackernews.com/2024/02/cisa-adds-solarwinds-web-help-desk-flaw.html
