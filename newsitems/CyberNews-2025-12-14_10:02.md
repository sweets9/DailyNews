**Critical RCE Vulnerability in React Ecosystem (React2Shell)**

A critical remote code execution (RCE) vulnerability, dubbed "React2Shell" (CVE-2025-55182), has been discovered in React Server Components (RSC) and affects multiple popular frameworks including Next.js, Waku, Vite, React Router, and RedwoodSDK. This flaw allows unauthenticated attackers to execute arbitrary code on the server with a single HTTP request, essentially a "no password required" attack. Multiple hacker groups are already actively exploiting this vulnerability to spread malware.

Why it Matters: Given the widespread use of React and Next.js in web development, this vulnerability poses a significant risk to a vast number of web applications. A successful exploit could lead to a full server compromise, data theft, or deployment of ransomware. Immediate patching is critical.

Why it Matters: The inclusion of these vulnerabilities in the KEV catalog indicates that they are being actively exploited by attackers.
