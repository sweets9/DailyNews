A critical remote code execution (RCE) vulnerability in HPE OneView (CVE-2025-37164) is being actively exploited. The flaw, with a CVSS score of 10.0, allows unauthenticated attackers to execute arbitrary code and take full control of affected systems. The vulnerability exists in an unsecured API endpoint. **Critical Flaw in SmarterMail Allows Remote Code Execution**
A critical vulnerability (CVE-2025-52691) in SmarterMail email servers allows unauthenticated attackers to upload arbitrary files and achieve remote code execution (RCE). The vulnerability has a CVSS score of 10.0. An attacker could exploit this flaw to take complete control of the email server, steal sensitive data, and launch further attacks. SmarterTools has released a patched version of SmarterMail.
Why it Matters: Email servers are high-value targets for attackers. This vulnerability is particularly dangerous because it is unauthenticated, meaning an attacker needs no prior access to exploit it. A compromised email server can lead to significant data breaches, business email compromise (BEC) attacks, and a loss of customer trust.
Link: https://thehackernews.com/2026/01/11/smartermail-critical-vulnerability-allows-unauthenticated-rce.html

**High-Severity Flaw in Veeam Backup & Replication**
Veeam has patched a high-severity vulnerability (CVE-2025-59470) in its Backup & Replication software that could allow for remote code execution. The flaw, with a CVSS score of 9.0, requires the attacker to have the "Backup Operator" or "Tape Operator" role. While no active exploitation has been reported, the vulnerability could be chained with other attacks to gain initial access and then escalate privileges. Veeam has released an update to address the issue.
Why it Matters: Backup and replication software is a critical component of any organization's disaster recovery and business continuity plan. A compromise of this system could allow an attacker to delete or encrypt backups, making it impossible to recover from a ransomware attack or other data loss event.
