**Critical RCE Flaw in React Server Components (React2Shell) Under Active Exploitation**
A critical remote code execution (RCE) vulnerability, dubbed "React2Shell" (CVE-2025-55182), is being actively exploited. The flaw, which carries the highest possible severity score (CVSS 10.0), affects React Server Components. Attackers can exploit the way React processes data sent from a browser to the server to execute arbitrary commands on the system. Compromises have been observed in various sectors, including construction and entertainment.
*Why it Matters:* This is a critical, unauthenticated vulnerability in a popular web development technology. Any organization using React Server Components is at high risk of complete system takeover. Immediate patching and investigation for signs of compromise are crucial.
*Link:* https://thehackernews.com/2025/12/critical-rsc-vulnerability.html

**Unpatched Zero-Day in Gogs Git Service Allows Remote Code Execution**
A high-severity, unpatched zero-day vulnerability (CVE-2025-8110) in the Gogs self-hosted Git service is being actively exploited. The flaw allows an attacker to overwrite files, leading to remote code execution. Over 700 instances are reportedly vulnerable. This vulnerability is a bypass for a previously patched issue, highlighting a persistent security risk in the software.
*Why it Matters:* Gogs is used by organizations to manage source code. A compromise could lead to theft of intellectual property, insertion of malicious code into software, and a launchpad for further attacks into the corporate network. Since it is a zero-day, there is no patch, and exploitation is ongoing.
*Link:* https://thehackernews.com/2025/12/unpatched-gogs-zero-day.html

**Attackers Exploiting Hard-Coded Keys in Gladinet Secure Storage Products**
Threat actors are actively exploiting hard-coded cryptographic keys in Gladinet's CentreStack and Triofox enterprise storage products. This allows attackers to gain unauthorized access, execute malicious code, and potentially decrypt sensitive data. The attacks chain this vulnerability with a previously disclosed flaw (CVE-2025-11371) to take full control of the server.
*Why it Matters:* These products are used for secure file sharing and collaboration. A successful attack can lead to a significant data breach, exposing confidential company files and customer information. Organizations using these products must update immediately and rotate all security keys.
*Link:* https://thehackernews.com/2025/12/gladinet-hard-coded-keys-exploitation.html

**WinRAR Vulnerability Actively Exploited to Hack Trading Accounts**
A path traversal vulnerability in the popular WinRAR file compression software (CVE-2025-6218) is being actively exploited by multiple threat groups. Attackers are using specially crafted `.RAR` or `.ZIP` archives to execute malicious code when the archive is opened. The U.S. Cybersecurity and Infrastructure Security Agency (CISA) has added the flaw to its catalog of known exploited vulnerabilities.
*Why it Matters:* WinRAR is a ubiquitous utility in enterprise environments. This attack vector is simple and relies on tricking a user into opening a malicious file. A successful compromise can lead to malware infection, data theft, and loss of control over user workstations.
*Link:* https://thehackernews.com/2025/12/winrar-vulnerability-actively-exploited.html

**Cyberattacks Target Oracle E-Business Suite, 100+ Organizations Compromised**
A sophisticated campaign has been targeting Oracle's E-Business Suite (EBS), a collection of enterprise resource planning (ERP), customer relationship management (CRM), and supply-chain management (SCM) applications. The attacks have reportedly compromised over 100 organizations globally by exploiting vulnerabilities in the widely-used business software.
*Why it Matters:* ERP systems are the backbone of an organization, managing finance, HR, and supply chain data. A breach of this system is catastrophic, leading to major financial fraud, theft of sensitive employee and customer data, and disruption of core business operations.
*Link:* https://cybersecuritynews.com/oracle-ebs-attacks/

**Google Issues Emergency Patch for Actively Exploited Chrome Zero-Day**
Google has released an urgent security update for the Chrome web browser to fix a zero-day vulnerability that is being actively exploited in the wild. This is the eighth such zero-day flaw discovered and patched in Chrome this year. Details of the vulnerability are being kept under wraps to allow users time to update.
*Why it Matters:* Chrome is the world's most popular web browser. A zero-day exploit allows attackers to compromise user machines simply by having them visit a malicious website, bypassing other security controls. All users and organizations should prioritize applying the update immediately.
*Link:* https://www.qualys.com/2025/12/11/google-chrome-zero-day/

**UK Introduces New Cyber Security and Resilience Bill**
The U.K. government has introduced a new bill to strengthen the security of the nation's essential services. The Cyber Security and Resilience Bill expands the scope of the previous Network and Information Systems (NIS) Regulations, applying stricter security and incident reporting rules to more organizations, including third-party service providers like Managed Service Providers (MSPs).
*Why it Matters:* This legislation signals a move towards holding more organizations accountable for cybersecurity, especially those that are part of the critical digital supply chain. Businesses in the UK, particularly in the tech services sector, will need to review their security practices to ensure compliance with the new, more stringent requirements.
*Link:* https://www.hackerone.com/resources/uk-cyber-security-and-resilience-bill
