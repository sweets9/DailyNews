**Critical SmarterMail Vulnerability Actively Exploited in Ransomware Attacks**

**Notepad++ Vulnerability Allows Arbitrary Code Execution**
A critical vulnerability, CVE-2025-15556, has been discovered in Notepad++ versions prior to 8.8.9. The vulnerability exists in the application's update process and could allow an attacker to execute arbitrary code. The issue stems from the updater not properly verifying the authenticity of the update package, which could be replaced with a malicious file.
Why it Matters: Notepad++ is a very popular text editor, and this vulnerability could be used to target a large number of users, including developers and system administrators. A successful exploit could lead to a full system compromise.
Link: https://www.yorku.ca/issue/1707206400/
